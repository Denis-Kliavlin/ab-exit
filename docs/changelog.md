## [v6.56_clean] — 2026-06-10

### Принципиальные изменения архитектуры

- **Новая структура репо.** Один большой монолит → 57 отдельных файлов. Каждый раздел = свой `.md` файл размером 5-15 KB.
- **Версионирование через имена файлов.** Не правка, а создание новых версий. `section_005_v1.md` → `section_005_v2.md` при обновлении.
- **Формат `.md` без конвертации в Google Doc.** Решает критический баг сломанного markdown (`\#` вместо `#`).
- **Workflow разделён между Claude в чате и Claude Code.** Claude пишет тексты разделов; Claude Code пушит на GitHub и делает массовые операции.

### Added

- `README.md` — описание архитектуры и принципов
- `INDEX.md` — карта 57 разделов с их статусом и именами файлов
- `CHANGELOG.md` — этот файл

### Status of section files

Из 57 разделов:
- **12 разделов 📝 Restored** — полные тексты в памяти Claude в чате (5, 11, 13, 14, 17, 23, 24, 29-32, 47)

## [v6.56] — 2026-06-10 (предыдущая попытка, deprecated)

### Сделано

- Применён mapping реорганизации из Master Plan
- 94 раздела v6.53 распарсены и реорганизованы по тематике
- Разделы 96, 97, 98 интегрированы
- Создан монолит `AB-EXIT_v6.56_FINAL_COMPLETE` (1.2 MB) и 10 split-файлов

### Известные проблемы (deprecated в этой ветке)

- ❌ **Markdown сломан во всех 10 split-файлах** — везде `\#` вместо `#`, `\*\*` вместо `**`. Заголовки не рендерятся.
- ❌ **Нумерация хаотична** — в файле 03 идут подряд `91 → 13 → 14 → 93`. Старые из v6.53 сохранили номера, новые получили новую нумерацию.
- ❌ **6 placeholder'ов отсутствуют** — разделы 5, 13, 14, 17, 24, 47 не имеют контента.
- ❌ **Файлы слишком большие для чтения через chat API** — 02 (193K), 04 (229K), 05 (160K), 07 (252K), 09 (150K) не помещаются в context window claude.ai.

### Решение проблем — переход на v6.56_clean

Все эти проблемы решены архитектурно в v6.56_clean (этот репо):
- Маленькие файлы → проблема context window решена
- `disableConversionToGoogleType: true` → markdown не ломается
- Сквозная нумерация в именах файлов → нумерация прозрачна
- Восстановлены тексты 6 placeholder'ов из памяти Claude в чате
